# `skyhigh`

目標管理アプリ

## アプリの概要

ToDOアプリと目標管理アプリを別個にするのをやめて1つのアプリにまとめる。
緊急ではないが、重要性の高いすべきことがちゃんとできるようなアプリを構想する。

### アプリの概要-⑴ログイン機能-

ユーザーごとに異なるメモの内容を保管しておくために、firebaseを用いる。
Authenticationでログインフォームを作成し、データはfirestoreに保管する。

### `アプリの概要-⑵メモの登録・削除機能￥`

メモは目標の大きさと遠さ別に管理する。
大きく遠い目標→例）CTOになる
小さく近い目標⇨例）Reduxを身につける

### `アプリの概要-⑶メモのひも付け機能`

大きな目標と小さな目標をひも付ける。
まだ取り組めていない目標がないかどうかがわかるように。
あるいは、目標が大きいばかりできっちりとしたステップを構想できていないかもしれない。
そういったチェック機能を備えたアプリにしたい。
